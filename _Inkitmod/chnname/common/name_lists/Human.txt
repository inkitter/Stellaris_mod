## 舰艇命名规则
#
# 护卫舰以小城市（市区非农人口在20万以下）命名，如“崇州”舰；
# 驱逐舰以大中城市，如“烟台”舰（市区非农人口在20万-100万）；
# 巡洋舰以省会城市与超特大城市，如济南舰（市区非农人口在100万以上）（不含直辖市）；
# 战列舰以行政省/行政区/直辖市命名，如“辽宁”舰（包含废弃的行省/行政区/行政署）；
#
# 科研船以中科院院士命名，如“邓稼先”号（1994年～2002年）；
# 工程船以古代科学家命名，如“毕升”号（参考《中国古代科学家传记 （上）》）；
# 殖民船以湖泊命名，如“抚仙湖”号；
# 运输舰以河流命名，如“松花江”号；
#
# 军事基地以山川命名，如“横断山”。
# 小型基地以海拔2000米以下的山脉命名，
# 中型基地以海拔2000米到5000米（不含）的山脉命名，
# 大型基地以海拔5000米以上的山脉，
# 边哨站以序号命名。
